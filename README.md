# PulsarFrame
An explot used for entertainment

## What Is It
PulseFrame is an Exploit that can allow you to open sites unblocked by using a data url generator to load a site that was previously blocked by a extention, or a school district.

## How to use
PulsarFrame can be used a couple of ways to use this.                                                                               
1. copy the link below and enter into a browser
2. copy the html and paste into a data url generator or a code sandbox
3. if you can, download the html code and open the link file.
